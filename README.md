# rustytele
Rustytele is converter for any kind of telegram sessions to grammers sessions.

## About
Rustytele supports:

- [&check;] - TData
- [&check;] - Telethon file
- [&check;] - Pyrogram file
- [&cross;] - Telethon string
- [&cross;] - Pyrogram string
- [&cross;] - Plain `dc_id:auth_key:ip:port` format
